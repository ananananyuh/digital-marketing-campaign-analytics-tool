# 📊 Digital Marketing Campaign Analytics Tool

An interactive and professional analytics dashboard built with Streamlit, Pandas, and Plotly for analyzing digital marketing campaign performance across multiple advertising platforms.

---

## 🚀 Overview

The Digital Marketing Campaign Analytics Tool helps marketers and analysts visualize, compare, and evaluate campaign performance across platforms such as:

- Google Ads
- Facebook Ads
- Instagram Ads

The dashboard provides real-time insights into campaign effectiveness, return on investment (ROI), revenue generation, conversion rates, and advertising spend.

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/ananananyuh/digital-marketing-campaign-analytics-tool.git
```

### 2. Navigate to Project Folder

```bash
cd digital-marketing-campaign-analytics-tool
```

### 3. Create Virtual Environment

```bash
python -m venv .venv
```

Activate Environment:

#### Windows

```bash
.venv\Scripts\activate
```

#### Mac/Linux

```bash
source .venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Start the Streamlit server:

```bash
streamlit run app.py
```

The application will launch in your browser at:

```text
http://localhost:8501
```
